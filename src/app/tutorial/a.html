In Melon Editor, the Game World is identified as "level". Level contains any game object like tile map, layer area, camera stage, sprite, and leaderboard that available to the player during the development mode.

A Game Level generally used to make a scenario of your game, increase / decrease the difficulty to appeal to players with different skill levels with any kind of quest. Each level presents new content and challenges to keep player's interest.

MelonIDE Level Editor

Melon Editor Level is divided into 3 parts, there are :

  • Level Properties
  • Level Layer
  • Level Stage

    Level Properties

  • Is the level component that use to manage the orientation of the Game, how the camera view showed on your game(Orthogonal / Isometric), set the column & Row that will appear on the tiles, set the size (width&Height) of your tiles, settings the game background and to show/hide the grid.

    • Level Layer

  • is the level component to add / remove a game object like sprite, tiles, etc. This component shows the list of all game objects in that level and sets up the Viewport and Collision.

    • Level Stage

  • is the area that shows and simulate the design of your level.

    Creating A Level

    To create a new level you can click on "Create" > "Level" at the menubar. After this window appears then fill it with the name of your level. After done press Create!

    Adding an object Into Level

    To import new object into level editor, you can go to Layer Level then click on the "+" button then choose one of any type object that you want. Melon Editor supports 5 types of Game Objects (Tile layer,Object Layer(Sprite), Images Layer, Text & Bitmap Text). All objects that have imported into level editor be you removed by clicking the "-" button,copy & duplicate it or rename by clicking the "rename" button.
