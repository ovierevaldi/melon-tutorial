In Facebook Instant Games, you can sell your in game items with In App Purchase. You can use In App Purchase feature in Melon Editor by using the code below.

First make sure you already configured your game and ready to launch to the Facebook Instant Game platform, if you don’t please read this Facebook Instant Game In App Purchase and the Instant Game API Documentation first.

Second, Enable Facebook Instant Game Plugin in the Plugin Tab above the melon editor

After enabling the plugin, write the code below somewhere in your game (after finished the level for example) and then showing the Ads for the player. There are two different type of Ads that can be showed in your app: Interstitial and Video Reward

  • Check Support In App Purchase

    You may want to check if your customer supported the IAP (In App Purchase) first before showing the products.

  • Get In App Purchase Catalog

    To showing the product that you sell to customer, you can call the getIAPCatalog() method to get the catalog list

  • Purchasing Item

    To purchasing the item, you can call purchaseIAP method, you will need your productID, and devPayload ( you can leave devPayload it blank if you want).

  • Get Purchases Items

    To get purchases item by the player, you can call getPurchasesIAP method and it will return you the list of item that already purchased by the player.

  • Consume Purchased Item

    To consume the item from the player, you can call consumePurchasedIAP() method, don’t call consumePurchasedIAP() If you want the item keeps in the player account. To use consumePurchasedIAP you will need the purchaseToken.
