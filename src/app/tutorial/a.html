In the concept of blockly programming, the smallest code component is called "Block". When the block component has a code, we call it "Block Code".
There are 3 types of blockcode:
  1. The first is a parcode block that can integrate several blockcodes
  2. Second type, blockcode with taq input is used as data / process input from blockcode that blocks the process of generating data.
  3. Third type, blockcode with an external arrow tag is used as a data / process input from blockcode after the process will be generated.
